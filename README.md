# projekt-ZMP

Repozytorium do projektu z przedmiotu Zaawansowane Metody Prognozowania. W projekcie należało zaprognozować rozwój rynku energetycznego w Polsce za pomocą mierników syntetycznych, które były prognozowane bezpośrednio oraz poprzez prognozy poszczególnych składowych tego miernika i agregację tych prognoz.
